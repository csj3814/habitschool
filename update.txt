📦 1단계: EXIF 검증 + 한국 타임존 + 갤러리 UI
========================================

[EXIF 메타데이터 검증]
- 문제: 사진 촬영 시간과 업로드 시간이 다를 때 부정 발생 가능
- 해결: EXIF.js 라이브러리로 촬영 시간 자동 추출 및 검증
- 결과: 사진의 실제 촬영 시간이 한국 시간 기준으로 검증됨

[한국 타임존 보정]
- 문제: 서버 시간과 로컬 시간 불일치
- 해결: KST (UTC+9) 계산 로직 추가
- 코드: `new Date(now.getTime() + (now.getTimezoneOffset() * 60000) + (9 * 60 * 60 * 1000))`

[갤러리 UI 개선]
- 사용자 이름 강조 (font-weight: bold, 색상 primary)
- 친구는 ⭐️ 아이콘으로 구분
- 날짜 포맷: YYYY-MM-DD → YYYY. MM. DD
