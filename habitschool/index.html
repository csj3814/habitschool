📹 10초 하이퍼랩스 쉽게 찍는 법

🍎 iPhone (iPhone 13~16 / iOS 17~18)

  1. 카메라 앱 열기 → 하단 모드에서 타임랩스 선택
  2. 휴대폰을 가슴 높이로 고정하고 이동 동선 먼저 확인
  3. 촬영 버튼 누르고 운동 동작 20~40초 촬영
  4. 종료 후 미리보기 확인 → 저장된 영상 업로드

🤖 Android (Galaxy S23~S25 / Pixel 8~9)

  1. 카메라 앱 → 더보기 또는 모드 → 하이퍼랩스/타임랩스
  2. 배속은 기본(자동) 또는 10x 권장
  3. 핸드폰 흔들림 줄이고 운동 20~40초 촬영
  4. 갤러리에서 재생 확인 후 업로드

💡 팁: 너무 짧게(3~5초) 찍으면 썸네일 품질이 떨어질 수 있어요. 최소 15초 이상 촬영을 권장합니다.
